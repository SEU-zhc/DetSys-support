About DetSys

DetSys is a mobile computer-vision application for road infrastructure inspection, supporting iOS 18.5+macOS 15.5+visionOS 2.5+. Built with Swift 5.0 and SwiftUI, it offers real-time road distress detection, multi-object tracking, GPS synchronization, and data upload.

核心功能

  • 智能检测：实时视频流处理，前/后置摄像头与 USB 摄像头
  • 多目标追踪：ByteTrack 算法，确保目标持续跟踪
  • 任务管理：巡检任务创建、模板与进度管理
  • 模型管理：远程下载、本地缓存、动态更新
  • 可视化界面：四区域布局，实时检测叠加显示

Key Features

  • Smart Detection: Real-time video processing with front/rear and USB cameras
  • Multi-Object Tracking: ByteTrack for persistent target association
  • Task Management: Creation, templates, and progress tracking
  • Model Management: Remote download, local caching, dynamic updates
  • Visualization: Four-pane UI with live overlay

技术架构（摘要）

  • 检测引擎：YOLO CoreML 模型（支持 8 类路面病害）
  • 跟踪算法：ByteTrack 多目标跟踪，避免重复计算
  • 定位系统：高精度 GPS 与检测结果实时同步
  • 数据流：摄像头输入 → YOLO 检测 → ByteTrack 追踪 → 后处理 → 数据上传
  • 性能与可靠性：线程池监控、延迟加载、防抖机制、失败任务自动重试（最多 50 次）

Architecture (Overview)

  • Detector: YOLO CoreML (8 road-distress classes)
  • Tracker: ByteTrack, minimizing redundant computation
  • Localization: High-accuracy GPS synchronized with detections
  • Pipeline: Camera → YOLO → ByteTrack → Post-processing → Upload
  • Perf & Reliability: Thread-pool monitoring, lazy loading, debounce, auto-retry (up to 50x)

是否支持离线检测？

检测模型在设备本地运行，可在无网络下执行检测；任务数据上传需网络连接。

支持哪些设备与系统？

Is offline detection supported?

隐私与数据

DetSys 以本地处理为优先，摄像头帧经模型推理后生成检测结果。除非用户在任务同步或云端备份中明确启用上传，否则原始图像和定位数据不会离开设备。

Privacy & Data

DetSys prioritizes on-device processing. Camera frames are inferred locally. Unless users explicitly enable sync/backup, raw images and location data remain on device.
